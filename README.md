Launch in dev

In one terminal 
```cd parcel-react```
```yarn install```
```yarn start```

In another terminal
```./gradlew bootRun```

(For hot reload use intellijIdea and lauch with it)

Build for production

```./gradlew build```

Run production jar 
```java -jar build/libs/demo-0.0.1-SNAPSHOT.jar```
